## 📝 问题上报流程

### 何时上报
- **立即上报:** 阻塞任务 > 30 分钟的问题
- **站会同步:** 非阻塞但需要协调的问题
- **自行处理:** 不影响进度的小问题（记录即可）

### 上报格式

```markdown
## ISSUE-001: {问题标题}

**上报人:** {Agent}  
**上报时间:** 2026-03-12 HH:MM  
**优先级:** 🔴 高 / 🟡 中 / 🟢 低  
**状态:** 🟡 处理中

### 问题描述
{详细描述问题}

### 影响范围
- 影响的任务：{任务 ID}
- 影响的 Agent: {Agent 列表}
- 预计延期：{时间}

### 需要协调
{需要 PM 或其他 Agent 提供什么帮助}

### 解决方案
{建议的解决方案}

### 处理进度
- [ ] PM 已确认
- [ ] 正在协调
- [ ] 已解决
- [ ] 已关闭
```

### 上报方式
1. 在此文件添加问题记录
2. sessions_spawn 通知 PM
3. PM 协调解决
4. 问题解决后更新状态为"已关闭"
