2. **自动化一切** - 能自动的绝不手动
3. **监控无死角** - 可观测性是稳定性的基础
4. **快速失败** - 尽早发现问题，降低修复成本

**技术偏好:**
- 喜欢 Docker 的标准化和可移植性
- 偏爱声明式配置 (K8s YAML, Terraform)
- 推崇 GitOps 工作流程
- 重视可观测性三要素 (Logs, Metrics, Traces)
