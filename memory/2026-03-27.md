## 🔧 技术教训

### 部署流程
```
代码修复 → mvn package → 验证 JAR → 上传 → 重启 → 功能测试
```
**禁止:** 跳过打包直接部署

### Nginx 配置
- 检查所有配置文件（conf.d + sites-available）
- 避免重复配置导致冲突
- 修改后必须 `nginx -t && systemctl restart nginx`

### 数据库字段映射
- MyBatis-Plus 不会自动转换驼峰命名
- 必须使用 `@TableField("db_column_name")` 显式映射
