## 🔧 使用方法

### 快速检查（2 分钟）

```bash
# 1. 检查所有 Agent 的 MEMORY.md
ls agent/workspace-*/MEMORY.md

# 2. 检查核心文件
for agent in guantang jiangrou dousha suancai; do
  echo "=== $agent ==="
  ls agent/workspace-$agent/{IDENTITY,ROLE,SOUL,MEMORY}.md
done

# 3. 检查 ARCHITECTURE.md 更新
git diff HEAD agent/ARCHITECTURE.md
```

### 深度检查（10 分钟）

1. 打开此检查清单
2. 逐项勾选每个项目
3. 发现任何问题立即修复
4. 修复后重新检查
5. 全部通过才算完成

### 自动检查（推荐）

**创建 PowerShell 脚本:**
```powershell
# check-completeness.ps1
$agents = @("guantang", "jiangrou", "dousha", "suancai")
$files = @("IDENTITY.md", "ROLE.md", "SOUL.md", "MEMORY.md", "TOOLS.md")

foreach ($agent in $agents) {
    Write-Host "=== Checking $agent ===" -ForegroundColor Cyan
    foreach ($file in $files) {
        $path = "agent/workspace-$agent/$file"
        if (Test-Path $path) {
            Write-Host "✅ $file" -ForegroundColor Green
        } else {
            Write-Host "❌ $file MISSING" -ForegroundColor Red
        }
    }
}
```
