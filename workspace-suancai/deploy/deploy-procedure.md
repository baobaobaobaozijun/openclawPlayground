## 2. 部署步骤

### 2.1 准备工作
1. 确保服务器连接正常 (已验证)
2. 确认部署目录 /opt/baozipu/ 可用 (已验证)
3. 验证 Docker Compose 配置 (已验证)

### 2.2 服务部署
1. 进入部署目录: `cd /opt/baozipu/`
2. 启动服务: `docker-compose up -d`
3. 检查服务状态: `docker-compose ps`

### 2.3 服务验证
1. 检查后端服务: `curl http://localhost:8081/health`
2. 检查前端服务: `curl http://localhost`
3. 验证数据库连接
4. 验证缓存连接

## 3. 回滚方案

### 3.1 紧急停止
```bash
cd /opt/baozipu/
docker-compose down
```

### 3.2 版本回退
```bash
# 停止当前服务
docker-compose down

# 恢复到上一版本的镜像
docker-compose pull --ignore-pull-failures
docker-compose up -d
```

### 3.3 数据库回滚
- 使用预先准备的数据库备份
- 执行备份恢复脚本

## 4. 注意事项

1. 当前 MySQL 和 Redis 仅监听本地地址 (127.0.0.1)，如需远程访问需调整配置
2. 端口 8081 尚未有服务监听，需确认后端服务启动状态
3. 部署前建议检查 UFW 防火墙配置，确保必要端口开放
