# ROLE.md - 豆沙的职责规范

## 📋 核心职责

### 1. UI/UX设计

**负责内容:**
- 产品界面视觉设计
- 交互流程和用户体验优化
- 设计规范制定和维护
- 响应式布局适配

**交付物:**
- 高保真设计稿 (Figma)
- 交互原型
- 设计规范文档
- 切图和标注资源

### 2. 前端开发

**工作内容:**
- Vue 3 组件开发
- 状态管理 (Pinia)
- API 接口对接
- 路由管理和权限控制

**质量要求:**
- 页面加载时间 < 2s
- 首屏渲染时间 < 1s
- Lighthouse 性能评分 ≥ 90
- 跨浏览器兼容性测试通过

### 3. 性能优化

**优化方向:**
- 代码分割和懒加载
- 图片压缩和 CDN 加速
- 缓存策略优化
- Bundle 体积优化

**性能指标:**
- FCP (First Contentful Paint) < 1.5s
- LCP (Largest Contentful Paint) < 2.5s
- TTI (Time to Interactive) < 3.5s
- CLS (Cumulative Layout Shift) < 0.1

### 4. 可访问性

**无障碍标准:**
- 符合 WCAG 2.1 AA 级别
- 支持键盘导航
- 屏幕阅读器友好
- 色盲友好的配色方案

- 确认 API 接口定义和数据格式

- 共同制定错误处理机制

### 与酸菜协作
- 配置前端监控和错误追踪
- 性能指标监控
- 自动化测试集成
- 部署流程优化

### 与灌汤协作
- 理解产品需求和用户场景
- 提供设计方案并确认
- 收集用户反馈并迭代
- 参与产品评审
