## 阶段 1：基础设施就绪（03-23 ~ 03-25）🔴 最高优先级

### T1-01: 修复 JAVA_HOME
**截止:** 03-23 18:00  

**具体操作：**
1. 查找 JDK 21 安装路径
2. 设置 Machine 级别环境变量 JAVA_HOME
3. 确认 PATH 包含 `%JAVA_HOME%\bin`

**验证命令（心跳检查会自动执行）：**
```powershell
java -version
# 预期输出包含: openjdk version "21.
echo $env:JAVA_HOME
# 预期输出: C:\...\jdk-21... (实际路径)
```

**输出：** 工作日志记录实际 JDK 路径和设置命令

### T1-02: 解决端口 8080 冲突
**截止:** 03-23 18:00

**具体操作：**
1. `netstat -ano | findstr :8080` 找到占用进程
2. 确认该进程是否可以停止
3. 停止占用进程，或修改后端配置使用其他端口

**验证命令：**
```powershell
netstat -ano | findstr :8080
# 预期：无输出（端口空闲）
# 或：后端 application.yml 中 server.port 改为其他值
```

### T1-03: 创建 MySQL 数据库
**截止:** 03-24 12:00

**具体操作：**
```sql
CREATE DATABASE IF NOT EXISTS openclaw DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER IF NOT EXISTS 'openclaw'@'localhost' IDENTIFIED BY '{密码}';
GRANT ALL PRIVILEGES ON openclaw.* TO 'openclaw'@'localhost';
FLUSH PRIVILEGES;
```

**验证命令：**
```powershell
mysql -u openclaw -p -e "SHOW DATABASES;" 2>&1 | findstr openclaw
# 预期输出: openclaw
```

**输出文件:** `F:\openclaw\code\deploy\scripts\init-database.sql`（上述 SQL 保存为脚本）

### T1-04: 启动 Redis 服务
**截止:** 03-24 12:00

**具体操作：**
1. 确认 Redis 已安装
2. 启动 Redis 服务
3. 设置开机自启

**验证命令：**
```powershell
redis-cli ping
# 预期输出: PONG

redis-cli info server | findstr redis_version
# 预期输出: redis_version:7.x
```

## 阶段 5：部署上线（04-02 ~ 04-04）

### T5-01: Docker Compose 配置
**截止:** 04-02 18:00  
**输出文件:** `F:\openclaw\code\deploy\docker-compose.yml`

**必须包含的服务：**
```yaml
services:
  mysql:
    image: mysql:8.0
    # 端口、数据卷、环境变量
  redis:
    image: redis:7
    # 端口、持久化
  backend:
    build: ../backend
    # 依赖 mysql, redis
  frontend:
    build: ../frontend
    # Nginx 静态文件服务
  nginx:
    image: nginx:alpine
    # 反向代理配置
```

### T5-02: Nginx 配置
**截止:** 04-02 18:00  
**输出文件:** `F:\openclaw\code\deploy\nginx\default.conf`

**必须包含：**

- API 反向代理（/api/ → backend:8080）
- gzip 压缩
- 缓存头
