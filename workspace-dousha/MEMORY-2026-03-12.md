## 遇到的问题

1. **vue-tsc 版本兼容性**
   - 问题：vue-tsc v1.8 与 Node.js v24 不兼容
   - 解决：暂时跳过类型检查，直接用 vite build
   - 影响：低

2. **Tailwind CSS v4 PostCSS 插件**
   - 问题：v4 版本将 PostCSS 插件移到独立包
   - 解决：安装 @tailwindcss/postcss 并更新配置
   - 影响：已解决
