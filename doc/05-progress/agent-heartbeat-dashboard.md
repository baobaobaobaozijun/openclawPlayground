## 🔔 心跳检查机制

### PM 职责
1. **定时检查** - 每 2 小时检查各 Agent 心跳
2. **看板维护** - 更新此看板记录心跳状态
3. **主动确认** - Agent 2 次未同步时主动联系
4. **站会通报** - Agent 3 次未同步时站会通报

### Agent 职责
1. **主动同步** - 收到心跳检查时主动 sessions_send 同步
2. **告知状态** - 包含任务 ID、名称、进度、状态
3. **告知任务** - 当前正在执行的具体工作
4. **问题上报** - 有阻塞或风险必须说明
