# Agent 进度汇报模板

**使用说明：** Agent 每 30 分钟主动汇报一次进度，使用此模板。

---

## 进度汇报格式

```markdown
【进度汇报 - {Agent 名} - {YYYY-MM-DD HH:mm}】

**当前任务：** {Plan-XXX RX 任务名称}
**进度：** {XX}%

**已完成：**
- [x] 子任务 1
- [x] 子任务 2

**进行中：**
- [ ] 子任务 3（预计 XX 分钟完成）

**待开始：**
- [ ] 子任务 4
- [ ] 子任务 5

**阻碍：**
- 无 / {具体阻碍及需要 PM 协调的事项}

**工作日志：** {F:\openclaw\agent\workinglog\{agent}\{文件名}.md}

**下次汇报：** {YYYY-MM-DD HH:mm}
```

---

## 汇报频率要求

| Agent 状态 | 汇报频率 | 说明 |
|-----------|---------|------|
| 执行中 | 每 30 分钟 | 任务进行中 |
| 阻塞 | 立即汇报 | 遇到阻碍时 |
| 完成 | 立即汇报 | 任务完成时 |
| 空闲 | 每 60 分钟 | 无任务时 |

---

## 违规处理

| 违规次数 | 处理措施 |
|---------|---------|
| 首次超时（>35 分钟） | 自动提醒 |
| 二次超时 | PM 警告 |
| 三次超时 | 记录违规，调整任务分配 |

---
