## 1. 用户认证流程

### 1.1 用户注册
- [ ] 访问注册页面 `/register`
- [ ] 输入有效用户名、邮箱、密码
- [ ] 提交注册表单
- [ ] 验证跳转至登录页面
- [ ] 验证数据库中用户记录创建

### 1.2 用户登录
- [ ] 访问登录页面 `/login`
- [ ] 输入用户名和密码
- [ ] 提交登录表单
- [ ] 验证跳转至首页
- [ ] 验证 NavBar 显示用户信息
- [ ] 验证 LocalStorage 中 Token 存储

### 1.3 用户登出
- [ ] 点击 NavBar 中的"退出"按钮
- [ ] 验证跳转至首页
- [ ] 验证 NavBar 显示登录按钮
- [ ] 验证 LocalStorage 中 Token 清除
