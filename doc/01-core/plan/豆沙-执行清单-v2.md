**问题：** 你自己分析出断点不统一（HomeView 992px, NavBar 768px）

  --bp-mobile: 576px;
  --bp-tablet: 768px;
  --bp-desktop: 992px;
  --bp-wide: 1200px;
}

**验证：** 所有媒体查询使用统一断点值，grep 检查无硬编码数字。

  { path: '/articles', component: ArticleListView },
  { path: '/articles/:id', component: ArticleDetailView },
  { path: '/admin/editor', component: ArticleEditorView, meta: { requiresAuth: true } },
  { path: '/admin/editor/:id', component: ArticleEditorView, meta: { requiresAuth: true } },
]
