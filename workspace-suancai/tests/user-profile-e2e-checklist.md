# 用户中心模块 E2E 测试用例清单
# Plan-016 R3 - E2E 测试

## 测试场景 1: 登录→访问个人中心→查看资料

**前置条件:**
- 用户已注册并激活账号
- 系统正常运行

**测试步骤:**
1. 打开登录页面
2. 输入正确的用户名和密码
3. 点击登录按钮
4. 点击导航栏中的"个人中心"链接
5. 查看页面显示的用户资料

**预期结果:**
- 成功登录系统
- 跳转到个人中心页面
- 页面正确显示用户的基本资料信息

## 测试场景 2: 编辑资料→保存→验证

**前置条件:**
- 用户已登录系统
- 位于个人中心页面

**测试步骤:**
1. 点击"编辑资料"按钮
2. 修改昵称、邮箱等信息
3. 点击"保存"按钮
4. 刷新页面或跳转到其他页面后再回到个人中心

**预期结果:**
- 资料修改成功
- 保存后页面提示"保存成功"
- 刷新后显示最新的资料信息

## 测试场景 3: 上传头像→预览→验证

**前置条件:**
- 用户已登录系统
- 位于个人中心页面

**测试步骤:**
1. 点击头像区域的"上传头像"按钮
2. 选择一张图片文件
3. 点击"确定"或"上传"按钮
4. 查看上传后的头像预览

**预期结果:**
- 头像上传成功
- 页面显示新的头像预览
- 保存后刷新页面，头像仍然显示正确
