# 服务器配置信息

**用途:** 博客系统部署目标服务器

---

## ⚠️ 服务器信息缺失

**当前状态:** 未配置远程服务器信息

**需要以下信息：**

| 配置项 | 说明 | 状态 |
|--------|------|------|
| 服务器 IP | 远程服务器公网 IP | ❌ 缺失 |
| SSH 端口 | 默认 22 | ❌ 缺失 |
| SSH 用户 | 登录用户名 | ❌ 缺失 |
| SSH 凭证 | 密码或私钥 | ❌ 缺失 |
| 操作系统 | Linux 发行版 | ❌ 缺失 |
| Docker | 是否已安装 | ❌ 未知 |
| MySQL | 是否已安装 | ❌ 未知 |
| Redis | 是否已安装 | ❌ 未知 |
| Nginx | 是否已安装 | ❌ 未知 |

---

## 临时方案（本地测试）

**在获得远程服务器信息前，使用本地环境测试部署：**

| 服务 | 地址 | 说明 |
|------|------|------|
| 后端 API | http://localhost:8080 | Spring Boot |
| 前端页面 | http://localhost/ | Nginx 或直接打开 dist/index.html |
| MySQL | localhost:3306 | 本地 MySQL |
| Redis | localhost:6379 | 本地 Redis（待安装） |
